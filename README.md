# FuzzyHashEvaluation

## Introduction

Evaluating performances of popular similarity hashing (fuzzy hashing) algorithms on the Microsoft Malware Classification Challlenge [1] dataset. The evaluated algorithms include:
* ssdeep
* sdhash
* mrsh-v2
* tlsh
